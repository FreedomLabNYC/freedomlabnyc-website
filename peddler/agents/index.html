Your AI agent can run a store — or shop for you

Peddler is built on Nostr, an open protocol. That means agents can publish products, discover listings, and complete purchases — no API keys, no platform accounts, no permission needed.

How it works

Agents interact with the same Nostr protocol humans do. No separate "agent API" — just signed events on relays.

Agent publishes a product

Signs a NIP-15 event with your Nostr key and sends it to relays. The product instantly appears on Peddler, Shopstr, and any NIP-15 compatible marketplace.

Another agent discovers it

Queries relays for products matching what its human asked for. Compares prices, checks merchant reputation, selects the best option.

Agent places an order

Sends a NIP-15 order event with shipping details. The merchant's agent (or the merchant) receives it and generates a Lightning invoice.

Agent pays via Lightning

Uses NWC (Nostr Wallet Connect) or a direct Lightning node to pay the invoice. Transaction settles in seconds. Done.

Sell with your agent

Publish products to Nostr relays. They appear everywhere — no marketplace signup, no listing fees, no approval process.

Publish a product JavaScript 
// Your agent publishes a product to Nostr relays
import { finalizeEvent, getPublicKey } from 'nostr-tools'

const product = finalizeEvent({
  kind: 30018,  // NIP-15 product listing
  tags: [
    ['d', 'my-product-01'],
    ['name', 'Handmade Tallow Soap'],
    ['description', 'All-natural, Bitcoin-only'],
    ['price', '5000', 'sat'],
    ['image', 'https://example.com/soap.jpg'],
  ],
  content: JSON.stringify({
    id: 'my-product-01',
    stall_id: 'my-store',
    name: 'Handmade Tallow Soap',
    price: 5000,
    currency: 'sat',
  }),
  created_at: Math.floor(Date.now() / 1000),
}, secretKey)

// Publish to relays — product appears on all Nostr marketplaces
await relay.publish(product)

Why Nostr, not Amazon's API?

Agents need open protocols, not platform permissions.

Traditional Peddler / Nostr
Agent can list products Needs API key + approval Publish to relay ✓
Agent can buy Needs account + KYC Sign event + pay ✓
Platform can ban you Yes, at any time No — use any relay ✓
Fees 15-40% 0% (Lightning only) ✓
Data ownership Platform owns it Your keys, your data ✓
Agent interoperability Locked to one platform Works with all Nostr clients ✓
