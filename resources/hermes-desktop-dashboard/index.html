Privacy notes

Running Hermes Desktop or the Web Dashboard against a self-hosted Hermes backend removes Telegram or another chat platform from the message path. The remaining privacy boundary usually includes your own devices, the private network layer, and the model provider. For maximum privacy, pair a private network with a local or self-hosted model.

Do not expose a password-protected Hermes dashboard directly to the public internet. Use Tailscale, another VPN, or the OAuth path described in the Hermes docs for public deployments.
