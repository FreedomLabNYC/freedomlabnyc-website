Share a Skill

Freedom Lab is moving skill sharing to the Telegram group. Create and polish your skill, then share it with the community for review and iteration.

How It Works

  1. Create the skill — Use the Skill Creator for AI Agents guide to write a clear, portable SKILL.md.
  2. Polish it — Remove private paths, secrets, local assumptions, and unnecessary bulk. Test any bundled scripts.
  3. Share in Telegram — Post the SKILL.md text for quick review, or attach the packaged .skill archive if it includes bundled resources.
  4. Iterate — Ask people using different agents to try it, then simplify and improve the skill based on feedback.

What to Post

  • A one-sentence summary of what the skill does.
  • Who the skill is for and what agent/tooling it was tested with.
  • The full SKILL.md contents, or a .skill attachment if scripts/references/assets are included.
  • Any setup requirements, API keys, commands, or safety notes people should inspect before use.

Make It Agent-Friendly

  • Keep the core instructions readable as plain Markdown.
  • Use agent-neutral language instead of assuming one specific agent.
  • Prefer portable commands and explain OS-specific alternatives.
  • Never share secrets, private data, raw credentials, or session logs.
